# Bookie Project
Projekt na zaliczenie przedmiotu Programowanie w języku Java s4.

Celem projektu było stworzenie kompleksowej aplikacji do zarządzania osobistą biblioteką książek - Bookie. Aplikacja stanowi nowoczesny system umożliwiający użytkownikom organizowanie swoich zbiorów książkowych, śledzenie postępów w czytaniu, ocenianie przeczytanych pozycji oraz dzielenie się opiniami z innymi czytelnikami w ramach społeczności.

_The purpose of the project was creating a complex app to manage private bookshelf. The app allows users to organize their books, track reading process, rating read position and sharing opinions with other readers._
